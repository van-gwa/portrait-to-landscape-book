# 四边木栏（蒸笼标签）+左中右

## 定义

该布局通过**四边木栏 + 蒸笼式左侧标签 + 左中右三列**的空间组织方式，实现角色/物品的详情展示与交互。属四边木栏系列大类，共 11 个 UI。

## 布局结构

```

│  │ [挂牌]                         │  │

│  │ │  │ │          │ │ [按钮]  │ │  │
│  │ │  │ │          │ │         │ │  │
```

## 三步判断法（基于11个UI图片分析）

### 第1步：看有没有四边木栏边框？

整个主操作区域四周是否被**深褐色木质纹理边框**包围？
- 是 → 继续（100%都满足）
- 否 → 不是本类型

这是最直观的特征——就像一幅装裱好的古画或窗框。

### 第2步：看左侧有没有蒸笼式竖向标签？

左侧是否有一列**竖向堆叠**的蒸笼造型导航按钮？
- "蒸笼"特征：竹蒸笼叠放造型，底部常有小火炉装饰
- 功能：用于切换不同子页面（如"基础/羁绊/天资/时装"）
- 垂直排列，通常2-4个

这是此类型的关键标志，蒸笼在**左侧**而非顶部。

### 第3步：看内部是不是左中右三列？

确认三列的具体内容分配：

| 列 | 位置 | 内容 | 特征 |
|----|------|------|------|
| **左** | 左侧窄列 | 蒸笼标签导航 + 挂牌 | 挂有木质名牌（如"门客""红颜""背包"） |
| **中** | 中央宽区 | 视觉展示（角色立绘/场景图/物品列表） | 常有圆窗背景或中式屏风 |
| **右** | 右侧宽区 | 详情信息面板 | 浅黄宣纸底+属性数值+操作按钮 |

三列齐全 → **四边木栏（蒸笼标签）+左中右** ✓

## 常见混淆类型及区分

### vs 四边木栏（蒸笼标签）+横列表

| 特征 | +左中右 | +横列表 |
|------|---------|---------|
| 中央区域 | 单一大图/角色展示 | 水平排列的多个条目卡片 |
| 右侧区域 | 详情面板（属性+按钮） | 无独立右区 |
| 布局逻辑 | 三列固定分区 | 单区横排列表 |

### vs T型-上标题-左-右

| 特征 | 四边木栏+左中右 | T型-上标题-左-右 |
|------|----------------|-----------------|
| 边框 | 有木栏边框 | 无木栏边框 |
| 左侧标签 | 蒸笼造型（竹编纹理） | 普通矩形标签 |

### vs 左标签+中图+右信息

| 特征 | 四边木栏+左中右 | 左标签+中图+右信息 |
|------|----------------|-------------------|
| 边框 | 有木栏边框 | 无边框 |
| 标签造型 | 蒸笼叠放式 | 普通矩形式 |

> **木栏边框画框装，蒸笼标签左列放。三列左中右分明，中展形象右详档。**

## 主要特征

- 四边木栏边框形成古画/窗框视觉效果
- 蒸笼标签在左侧竖向排列，非顶部
- 左中右三列各司其职：导航-展示-详情
- 中央区常有圆窗/屏风等中式元素
- 右侧面板常用浅黄宣纸背景

## 横竖屏适配要点

- 左中右三列在横屏时需重新分配宽度
- 中央展示区在横屏时更宽，可容纳更大角色图
- 右侧详情面板需保持可读宽度
