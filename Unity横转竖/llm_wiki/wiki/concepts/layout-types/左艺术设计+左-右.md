# 左艺术设计+左-右

## 定义

「左艺术设计+左-右」是艺术设计系列的基础类型——**左侧带有彩色艺术装饰元素（祥云/流苏/梅花/卷轴）+ 内部左右分栏**。共14个UI。

**核心判据**：左侧艺术装饰标签栏（非普通矩形标签）+ 内部左右分栏内容。

## 布局结构

```
┌──────────────────────────────┐
│ [返回]     [标题]           │
├──艺术装饰──┬────────────────┤
│            │                │
│ 祥云/梅花 │  左栏  │ 右栏  │
│ /流苏/    │  内容  │ 内容  │
│ 卷轴      │        │       │
│            │                │
└────────────┴────────────────┘
```

## 三步判断法（5样本+2验证，盲测2/2通过）

### 第1步：左侧有没有艺术装饰？

左侧标签/边框是否有以下艺术装饰元素？
- **祥云图案**（云朵造型装饰）
- **梅花/花卉**点缀
- **流苏**挂坠
- **卷轴**/立轴造型
- **彩色底纹**（非纯木纹）

有 → 继续。无 → 不是艺术设计系列。

### 第2步：内部是不是左右分栏？

艺术装饰框内是**左右分栏**结构？
- 是 → 继续
- 否 → 考虑其他艺术设计类型

### 第3步：排除其他艺术设计类型

- 左右艺术设计+一体 → 内部无分栏
- 左右艺术设计+横列表 → 内部横列表
- 左右艺术设计+竖列表 → 内部竖列表
- 本类型 → **左艺术设计+左-右** ✓

## 常见混淆

### vs 左-右

| 特征 | 左艺术设计+左-右 | 左-右 |
|------|----------------|-------|
| 左侧装饰 | **彩色艺术装饰** | 无装饰或纯功能标签 |
| 艺术元素 | 祥云/梅花/流苏/卷轴 | 无 |

### vs 四边木栏+左中右

| 特征 | 左艺术设计+左-右 | 四边木栏+左中右 |
|------|----------------|-----------------|
| 边框 | 彩色艺术装饰 | 纯木纹四边框 |
| 列数 | 左右两栏 | 左中右三栏 |
