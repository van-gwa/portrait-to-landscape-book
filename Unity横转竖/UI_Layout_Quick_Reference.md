### 记住这个黄金公式
```

前10种 = 约70%

→ 掌握前20种就能应付大多数情况！
```

### 记住这些高频词
- **标题** - 上面的标题行
- **列表** - 竖(纵向)/横(横向)
- **标签** - 左侧的分类标签
- **条** - 横条形的项目
- **弹框** - 小的浮窗
- **木栏** - 黄棕色边框（古风）
- **艺术设计** - 彩色装饰边框

## 使用这个表的方式

### 方式1: 直接按名称找
- 如果你知道大概的类型名称，直接Ctrl+F搜索
- 看关键特征确认
- 点击笔记链接查看具体例子

### 方式2: 按关键词找
- 有竖列表+标题+底部信息？ → `上标题-中竖列表-下信息`

- 有木栏边框？ → 搜索"四边木栏"系列
- 小弹框？ → 搜索"弹框"系列

- 简单类型不确定？ → 按简单类型分组找
- 中等复杂？ → 按中等复杂分组找
