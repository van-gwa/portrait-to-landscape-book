# 左-右

## 定义

该布局是左右布局中最基础的类型——**简单左右两栏分割**，无装饰边框（区别于四边木栏系列）、无特殊艺术设计（区别于艺术设计系列）、无左侧标签导航（区别于左标签系列）。共 36 个 UI。

**命名含义**：纯左右分栏，左区展示/选择，右区详情/操作。

## 布局结构

```
┌──────────────────────────────┐
│      [标题 - 可选]           │
├──────────┬───────────────────┤
│          │                   │
│  左区    │     右区         │
│  (展示/  │    (详情/操作)    │
│  选择)   │                   │
│          │                   │
└──────────┴───────────────────┘
```

## 三步判断法（基于36个UI图片分析）

### 第1步：看有没有左右分栏？

界面是否被一条**垂直分割线**分为左右两部分？
- 有 → 继续
- 否 → 不是本类型

### 第2步：排除带装饰/特殊元素的类型

是否存在以下特征？
- **四边木栏边框** → 是「四边木栏」系列，非本类型
- **彩色艺术装饰框** → 是「艺术设计」系列，非本类型
- **左侧蒸笼/挂牌标签导航** → 是「左标签」系列，非本类型
- **都没有** → 继续

### 第3步：确认左右各是什么？

| 列 | 典型内容 | 常见形态 |
|----|---------|---------|
| **左** | 展示/选择区 | 建筑缩略图、角色立绘、排行榜领奖台、物品图标、场景地图、可选列表 |
| **右** | 详情/操作区 | 属性数值、文字描述、奖励列表、操作按钮、排名列表 |

左右分栏 + 无装饰元素 → **左-右** ✓

## 五种常见子类型

### A. 左展示-右详情（约50%，最常见）

左侧为图片/立绘/场景展示，右侧为详细信息+按钮。
**典型UI**：UI_WorldBuild、UI_WorldBuildFinish、UI_MarketActive、UI_SilkroadGuestInfo、UI_TechnoCupInfo、UI_TradingPortChangeShip

### B. 左排行-右列表（约15%）

左侧为排行榜前三领奖台，右侧为完整排名列表。
**典型UI**：UI_ExpandRank、UI_BillionairesRank、UI_RankActTwo

### C. 左物品-右途径（约10%）

左侧为物品图标+名称，右侧为获取途径列表。
**典型UI**：UI_GetPaths、UI_ExpGetPath

### D. 左列表-右详情（约15%）

左侧为可选列表，右侧为选中项详情。
**典型UI**：UI_BeautyCompEnter、UI_CombinePersonnelGuest、UI_GuestEquipMount、UI_Notice

### E. 左信息-右操作（约10%）

左侧为数值/进度/奖励预览，右侧为操作入口。
**典型UI**：UI_BattlePass、UI_DailySign、UI_Fund、UI_Billionaires

## 常见混淆类型及区分

### vs 左广告图-右列表

| 特征 | 左-右 | 左广告图-右列表 |
|------|-------|----------------|
| 左区内容 | 展示图/立绘/缩略图（功能性） | 大幅广告图/宣传图 |
| 右区 | 详情+按钮 | 纯列表 |
| 用途 | 功能操作 | 广告展示+列表入口 |

### vs 四边木栏（蒸笼标签）+左中右

| 特征 | 左-右 | 四边木栏+左中右 |
|------|-------|----------------|
| 边框 | 无 | 四边木栏框 |
| 列数 | **2列**（左右） | **3列**（左中右） |
| 左侧标签 | 无 | 蒸笼标签 |

### vs 左艺术设计+左-右

| 特征 | 左-右 | 左艺术设计+左-右 |
|------|-------|-----------------|
| 装饰 | 无 | 彩色艺术装饰框 |
| 视觉风格 | 简洁 | 华丽 |

### vs T型-上标题-左-右

| 特征 | 左-右 | T型-上标题-左-右 |
|------|-------|-----------------|
| 顶部标题 | 可选，不贯穿 | **贯穿全宽**的T型标题栏 |
| 结构 | 二分 | 上横杠 + 下二分（T字） |

## 判断口诀

> **一竖分两半，左右各一半。无框无装饰，纯分栏最简。**

## 主要特征

- 最基础的左右分栏，无装饰、无特殊框
- 左区主展示/选择，右区主详情/操作
- 标题可选，不一定存在
- 与展饰类（木栏/艺术设计/左标签）的关键区别：**纯分栏，无额外部件**
