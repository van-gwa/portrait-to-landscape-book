# 展示-横条-左右对比

## 定义

「展示-横条-左右对比」是展示横条系列中的**升级前后对比**变体——横向长条弹窗 + 左右两列对比（升级前 vs 升级后）。全为Succ类（升级/提升成功）结尾。共11个UI。

## 布局结构

```
┌──────────────────────────────┐
│       全屏遮罩               │
│  ┌──────────────────────┐   │
│  │   [升级成功！]       │   │
│  │                      │   │
│  │ ┌────────┬─────────┐ │   │
│  │ │ 升级前 │ → 升级后│ │   │ ← 左右对比
│  │ │ Lv.5   │  Lv.6  │ │   │
│  │ │ 属性A  │ 属性A' │ │   │
│  │ └────────┴─────────┘ │   │
│  └──────────────────────┘   │
└──────────────────────────────┘
```

## 三步判断法（4样本+1验证，通过）

### 第1步：是不是横向长条 + 遮罩？

- 全屏遮罩 + 横向长条弹窗 → 继续

### 第2步：内部是不是左右对比结构？

- 左右两列，左侧=升级前/当前，右侧=升级后/目标
- 中间常有绿色箭头表示提升方向 → 继续

### 第3步：命名是否为 Succ 类？

- 11个UI全以 Succ/Change/Result 等结尾 → **展示-横条-左右对比** ✓

## vs 展示-横条-左图-右信息

| 特征 | 左右对比 | 左图-右信息 |
|------|---------|-----------|
| 左区 | 升级前数据 | 图片/立绘 |
| 右区 | 升级后数据 | 信息文字 |
| 对比箭头 | 常见 | 无 |

## 判断口诀

> **升级成功横条见，左旧右新绿箭连。Succ结尾皆此类，左右对比一目然。**
