# 弹框类

## 说明

该大类包含 **4 种布局**，共 **155 个 UI**。

## 包含的布局类型

| 布局名称 | UI 数量 |
|---------|--------|
| [确认弹框](llm_wiki/wiki/concepts/layout-types/确认弹框) | 87 |
| [活动打脸弹窗](llm_wiki/wiki/concepts/layout-types/活动打脸弹窗) | 30 |
| [使用弹框](llm_wiki/wiki/concepts/layout-types/使用弹框) | 24 |
| [功能开启&展示](llm_wiki/wiki/concepts/layout-types/功能开启&展示) | 14 |


**总计**：4 种布局，155 个 UI

## 大类特征

弹框类类型的布局具有以下共同特征：
- 统一的视觉风格和交互逻辑
- 相似的空间分区方式
- 一致的信息展示策略
