# 全屏类

## 说明

该大类包含 **5 种布局**，共 **21 个 UI**。

## 包含的布局类型

| 布局名称 | UI 数量 |
|---------|--------|
| [全屏背景界面](llm_wiki/wiki/concepts/layout-types/全屏背景界面) | 12 |
| [登录加载全屏](llm_wiki/wiki/concepts/layout-types/登录加载全屏) | 3 |
| [全屏获得新奖励](llm_wiki/wiki/concepts/layout-types/全屏获得新奖励) | 3 |
| [全屏入口](llm_wiki/wiki/concepts/layout-types/全屏入口) | 1 |
| [灯笼标签+彩带+全屏](llm_wiki/wiki/concepts/layout-types/灯笼标签+彩带+全屏) | 2 |


**总计**：5 种布局，21 个 UI

## 大类特征

全屏类类型的布局具有以下共同特征：
- 统一的视觉风格和交互逻辑
- 相似的空间分区方式
- 一致的信息展示策略
