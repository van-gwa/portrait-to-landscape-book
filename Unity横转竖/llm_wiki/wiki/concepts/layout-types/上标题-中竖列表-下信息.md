# 上标题-中竖列表-下信息

## 定义

该布局通过**上标题-中竖列表-下信息**的空间组织方式，实现内容的有效展示和交互。

## 布局结构

```
┌─────────────────────┐
│      标题            │
├─────────────────────┤
│  [Item 1]          │
│  [Item 2]          │ ↕ 可竖向滚动
│  [Item 3]          │
│  ...               │
├─────────────────────┤
│      信息            │
└─────────────────────┘
```

## 主要特征

- 清晰的空间分区方式
- 合理的内容排列逻辑
- 上标题系列类型的特征表现
