# 展示-横条-横列表

## 定义

「展示-横条-横列表」是展示横条系列中的**横条内嵌横向排列条目**变体——横条弹窗 + 内部物品/条目水平排列。全为 Succ/Tip/Result 类结尾。共7个UI。

**与「展示-横条-条内信息」的区别**：条内信息是垂直堆叠纯文字，本类型是横条内横向排列物品图标。

## 布局结构

```
┌──────────────────────────────┐
│       全屏遮罩               │
│  ┌──────────────────────┐   │
│  │  [标题：升级成功！]  │   │
│  │                      │   │
│  │ [图标1] [图标2] [..] │   │ ← 横向排列条目
│  │                      │   │
│  └──────────────────────┘   │
└──────────────────────────────┘
```

## 三步判断法（4样本+1验证，通过）

### 第1步：是不是横条弹窗 + 遮罩？

- 全屏遮罩 + 横向长条弹窗 → 继续

### 第2步：内部是不是横向排列条目？

- 物品图标/条目水平排列（左右滚动）→ 继续
- 注意：单条目时容易被误判，看命名后缀 Succ/Tip/Result

### 第3步：命名是否 Succ/Tip/Result？

- 是 → **展示-横条-横列表** ✓

---

## ⚠️ 陷阱：单条目视图

仅一个物品时看起来像单条展示，但仍属本类型（UI_BeautyPropsUseSucc）。

## 判断口诀

> **横条遮罩升级弹，内部图标横向展。Succ Tip结果类，七个小众此型完。**
