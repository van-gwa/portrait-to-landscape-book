# 艺术设计系列

## 说明

该大类包含 **4 种布局**，共 **24 个 UI**。

## 包含的布局类型

| 布局名称 | UI 数量 |
|---------|--------|
| [左艺术设计+左-右](llm_wiki/wiki/concepts/layout-types/左艺术设计+左-右) | 14 |
| [左右艺术设计+横列表](llm_wiki/wiki/concepts/layout-types/左右艺术设计+横列表) | 3 |
| [左右艺术设计+一体](llm_wiki/wiki/concepts/layout-types/左右艺术设计+一体) | 4 |
| [左右艺术设计+竖列表](llm_wiki/wiki/concepts/layout-types/左右艺术设计+竖列表) | 3 |


**总计**：4 种布局，24 个 UI

## 大类特征

艺术设计系列类型的布局具有以下共同特征：
- 统一的视觉风格和交互逻辑
- 相似的空间分区方式
- 一致的信息展示策略
