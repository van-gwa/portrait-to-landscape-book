# 红框方形玩法入口

## 定义

「红框方形玩法入口」是**活动主界面**——带有红色/红褐色装饰边框的全屏或大半屏活动弹窗，右下角通常有醒目的「进入活动」按钮。属一体框系列大类，共15个UI。

**核心判据**：红色装饰边框 + 活动主题视觉 + 进入活动按钮。

## 布局结构

```
┌─红色装饰边框──────────────┐
│  [返回]    活动标题        │
│                            │
│    ┌──────────────────┐    │
│    │  活动主题视觉     │    │
│    │  (转盘/地图/..)  │    │
│    └──────────────────┘    │
│                            │
│    [规则]  [奖励]         │
│                   ●       │
│              [进入活动]   │  ← 圆形入口按钮
└────────────────────────────┘
```

## 三步判断法（5样本+1验证，盲测1/1通过）

### 第1步：有红色/红褐色装饰边框？

- 界面四周有红色/红褐色装饰边框 → 继续
- 无 → 不是本类型

### 第2步：是活动主界面？

- 全屏或大半屏活动主题界面（非弹窗确认，非功能面板）
- 有活动视觉主题（转盘/地图/关卡图等）
- 是 → 继续

### 第3步：有「进入活动」入口按钮？

- 右下角通常有醒目的「进入活动」或类似入口按钮
- 是 → **红框方形玩法入口** ✓

> **红褐边框方方正，全屏活动气势盛。进入活动圆按钮，玩法入口由此进。**
