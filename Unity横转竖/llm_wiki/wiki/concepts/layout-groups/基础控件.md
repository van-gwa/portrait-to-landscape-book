# 基础控件

## 说明

该大类包含 **4 种布局**，共 **147 个 UI**。

## 包含的布局类型

| 布局名称 | UI 数量 |
|---------|--------|
| [tips](llm_wiki/wiki/concepts/layout-types/tips) | 100 |
| [panel](llm_wiki/wiki/concepts/layout-types/panel) | 26 |
| [mainUI](llm_wiki/wiki/concepts/layout-types/mainUI) | 8 |
| [panel+入口](llm_wiki/wiki/concepts/layout-types/panel+入口) | 13 |


**总计**：4 种布局，147 个 UI

## 大类特征

基础控件类型的布局具有以下共同特征：
- 统一的视觉风格和交互逻辑
- 相似的空间分区方式
- 一致的信息展示策略
