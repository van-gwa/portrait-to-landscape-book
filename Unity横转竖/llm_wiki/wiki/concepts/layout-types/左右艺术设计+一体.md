# 左右艺术设计+一体

## 定义

该布局通过**左右艺术设计+一体**的空间组织方式，实现内容的有效展示和交互。

## 布局结构

```
┌──────────┬────────────┐
│          │            │
│   左区   │   右区     │
│          │            │
│          │            │
└──────────┴────────────┘
```

## 主要特征

- 清晰的空间分区方式
- 合理的内容排列逻辑
- 艺术设计系列类型的特征表现

## 包含的 UI（共 4 个）

- [[prefab_final_analysis/UI_ChildTravelFailTip|UI_ChildTravelFailTip]]
- [[prefab_final_analysis/UI_WorldWonders|UI_WorldWonders]]
- [[prefab_final_analysis/UI_TimerTravelInfo|UI_TimerTravelInfo]]


## 横竖屏适配要点

- 布局在横竖屏转换时需要调整各区域的宽高比
- 关键信息应优先保证可见性
- 可根据屏幕方向调整内容的流向方式
