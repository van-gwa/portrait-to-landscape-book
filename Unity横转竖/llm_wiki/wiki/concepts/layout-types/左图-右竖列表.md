# 左图-右竖列表

## 定义

该布局通过**左图-右竖列表**的空间组织方式，实现内容的有效展示和交互。

## 布局结构

```
┌─────────────────────┐
│  [Item 1]          │
│  [Item 2]          │ ↕ 可竖向滚动
│  [Item 3]          │
│  ...               │
└─────────────────────┘
```

## 主要特征

- 清晰的空间分区方式
- 合理的内容排列逻辑
- 左右布局类型的特征表现

## 包含的 UI（共 1 个）

- [[prefab_final_analysis/UI_TopLeagueRank|UI_TopLeagueRank]]


## 横竖屏适配要点

- 布局在横竖屏转换时需要调整各区域的宽高比
- 关键信息应优先保证可见性
- 可根据屏幕方向调整内容的流向方式
