### 🔍 查询「四边木栏+左右-宽列表」
```

    → 对比表（对比其他 3 种类型）

```

### 🛠 分类新 UI 时的完整流程
```
1. 运行 classify-ui-type 自动分类脚本
2. 查看脚本的"节点分布特征"输出
3. 对比《快速指南》中的对比表和检验清单
4. 如果不确定，打开《四边木栏+左右-宽列表》详情页
5. 在 Unity Editor 中目视确认
```

### 📚 分级学习路线
```
【初学者】
1. 阅读《快速识别指南》的"常见错误案例"部分
2. 记住"三句话"法则
3. 了解基本的对比特征

【中级使用者】
1. 学习《快速指南》的完整对比表
2. 使用 4 大标志检验清单
3. 了解同类型的其他 6 种子类型

【高级管理者】
1. 读 SCHEMA 规范了解 Wiki 结构
2. 学习三个工作流（Ingest、Query、Lint）
3. 建立新的类型页面和对比指南
```
