# 灯笼标签+彩带+全屏

## 定义

该布局通过**灯笼标签+彩带+全屏**的空间组织方式，实现内容的有效展示和交互。

## 布局结构

```
┌──────────────────────────┐
│                          │
│                          │
│     全屏显示区           │
│     (背景/内容)          │
│                          │
│                          │
└──────────────────────────┘
```

## 主要特征

- 清晰的空间分区方式
- 合理的内容排列逻辑
- 全屏类类型的特征表现

## 包含的 UI（共 2 个）

- [[prefab_final_analysis/UI_BanquetAdvancedList|UI_BanquetAdvancedList]]
- [[prefab_final_analysis/UI_BanquetNormalList|UI_BanquetNormalList]]


## 横竖屏适配要点

- 布局在横竖屏转换时需要调整各区域的宽高比
- 关键信息应优先保证可见性
- 可根据屏幕方向调整内容的流向方式
