# 横列表

## 定义

该布局通过**横列表**的空间组织方式，实现内容的有效展示和交互。

## 布局结构

```
┌──────────────────────────┐
│[Item] [Item] [Item] ...│ → 可横向滚动
└──────────────────────────┘
```

## 主要特征

- 清晰的空间分区方式
- 合理的内容排列逻辑
- 一体框系列类型的特征表现

- 布局在横竖屏转换时需要调整各区域的宽高比
