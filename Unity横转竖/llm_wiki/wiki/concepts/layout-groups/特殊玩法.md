# 特殊玩法

## 说明

该大类包含 **4 种布局**，共 **19 个 UI**。

## 包含的布局类型

| 布局名称 | UI 数量 |
|---------|--------|
| [战斗](llm_wiki/wiki/concepts/layout-types/战斗) | 9 |
| [赛程](llm_wiki/wiki/concepts/layout-types/赛程) | 3 |
| [横条-VS](llm_wiki/wiki/concepts/layout-types/横条-VS) | 5 |
| [多列](llm_wiki/wiki/concepts/layout-types/多列) | 2 |


**总计**：4 种布局，19 个 UI

## 大类特征

特殊玩法类型的布局具有以下共同特征：
- 统一的视觉风格和交互逻辑
- 相似的空间分区方式
- 一致的信息展示策略
