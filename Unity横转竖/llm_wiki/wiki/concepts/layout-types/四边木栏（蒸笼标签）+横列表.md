# 四边木栏（蒸笼标签）+横列表

## 定义

该布局通过**四边木栏 + 蒸笼式侧标签 + 横向排列条目**的空间组织方式，实现多条目内容的横向浏览。属四边木栏系列大类，共 10 个 UI。

## 布局结构

```
┌──────────────────────────────────────┐
│  [返回]                    [货币栏]  │ ← 顶部状态栏
│  ┌─四边木栏边框───────────────────┐  │
│  │ [挂牌]                         │  │
│  │ ┌──────────────────────────┐   │  │
│  │ │ [卡片1] [卡片2] [卡片3]→│   │  │ ← 横向排列，可左右滚动
│  │ └──────────────────────────┘   │  │
│  │                          ↑     │  │
│  │                       横列表   │  │
│  │ [排序]           [操作按钮]    │  │ ← 底部功能栏
│  └────────────────────────────────┘  │
└──────────────────────────────────────┘
```

## 三步判断法（基于10个UI图片分析）

### 第1步：看有没有四边木栏边框？

主操作区域是否被**深褐色木质纹理边框**包围？
- 有 → 继续（绝大多数满足）
- 无但有蒸笼/挂牌元素 → 也可能是（如 UI_ActApexPvpProgress 用羊皮纸框）

### 第2步：看条目是否是横向排列？

内容区内的条目是否**水平方向排列**（可左右滚动）？
- 条目是横向展开的卡片/角色卡，非纵向列表
- 通常只有部分条目可见，暗示可滑动
- 每个条目通常是竖向卡片（角色头像+名字/属性）

**关键区分**：竖列表 = 上下滚动，横列表 = 左右滚动

### 第3步：看左侧是否有挂牌/蒸笼标签？

左侧是否有悬挂的木质标牌或蒸笼风格标签？
- 常见形式：挂牌上书功能名（如"红颜""门客""我的船队""文物商店"）
- 与"左中右"类型的区别：本类型**只有挂牌标签，没有三列分区**

三者齐备 → **四边木栏（蒸笼标签）+横列表** ✓

## 常见混淆类型及区分

### vs 四边木栏（蒸笼标签）+左中右

这是最容易混淆的两个类型！

| 特征 | +横列表 | +左中右 |
|------|---------|---------|
| 内容区结构 | **单一区**：条目横向排列 | **三列**：左导航-中展示-右详情 |
| 条目排列 | 水平排列（左右滚动） | 中央单一大图（非列表） |
| 右侧详情面板 | 无 | 有（属性+按钮） |
| 挂牌位置 | 左侧悬挂 | 左上挂牌+左侧蒸笼叠放 |

**判断核心**：看内容区有几个列？
- 一个区 + 里面横排卡片 → +横列表 ✓
- 三个区（左中右）→ +左中右 ✓

### vs 横列表（一体框系列）

| 特征 | 四边木栏+横列表 | 横列表（一体框） |
|------|----------------|-----------------|
| 边框 | 四边木质边框 | 一体框（无木纹） |
| 挂牌 | 左侧有挂牌/蒸笼标签 | 无 |

### vs 上标题-中横列表-下信息

| 特征 | 四边木栏+横列表 | 上标题-中横列表-下信息 |
|------|----------------|-----------------------|
| 木栏边框 | 有 | 无 |
| 标题 | 挂牌（侧边） | 顶栏文字标题 |
| 底部 | 功能按钮 | 底部信息条 |

## 子类型细分

根据10个UI的分析，可细分为：

### 类型A：角色/红颜选择型（约50%）

条目是角色卡+名字，用于选择角色查看详情或操作。

**典型UI**：UI_Beauty、UI_Guest、UI_RoleBeautyTip、UI_RoleGuestsTip、UI_StarWorksChoiceBeauty

### 类型B：商店/商品型（约30%）

条目是商品卡，用于展示可购买物品。

**典型UI**：UI_GachaCoinShop、UI_MuseumGachaShop

### 类型C：船队/队伍管理型（约10%）

**典型UI**：UI_TradingPortFleet

### 类型D：赛程/活动型（约10%）

**典型UI**：UI_ActApexPvpProgress、UI_YardMeet

## 判断口诀

> **木栏边框挂牌挂，条目横向排成画。左右滑动选目标，没有三列单区大。**

## 主要特征

- 四边木栏边框形成古画展示框
- 左侧悬挂木质名牌/蒸笼式标签
- 内容区为横向排列的条目（可左右滑动）
- 与"+左中右"的关键区别：单一内容区 vs 三列分区
- 底部通常有排序/筛选/操作按钮

## 横竖屏适配要点

- 横屏时横向列表可展示更多条目，减少滚动需求
- 木栏边框在横屏时需调整宽度适配
- 条目卡片在横屏时可增大尺寸提升视觉
