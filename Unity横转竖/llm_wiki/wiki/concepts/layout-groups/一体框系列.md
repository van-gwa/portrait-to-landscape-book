# 一体框系列

## 说明

该大类包含 **3 种布局**，共 **50 个 UI**。

## 包含的布局类型

| 布局名称 | UI 数量 |
|---------|--------|
| [一体框](llm_wiki/wiki/concepts/layout-types/一体框) | 22 |
| [红框方形玩法入口](llm_wiki/wiki/concepts/layout-types/红框方形玩法入口) | 15 |
| [横列表](llm_wiki/wiki/concepts/layout-types/横列表) | 13 |


**总计**：3 种布局，50 个 UI

## 大类特征

一体框系列类型的布局具有以下共同特征：
- 统一的视觉风格和交互逻辑
- 相似的空间分区方式
- 一致的信息展示策略
