# 展示-横条-左图-右信息

## 定义

「展示-横条-左图-右信息」是结算/结果展示类弹窗——**横向长条面板 + 左侧图片/立绘/展示 + 右侧文字/数据信息**。属展示横条系列大类，共15个UI，全为Succ/Result（升级/结算/获得）结尾。

**与「展示-横条-条内信息」的核心区别**：本类型是**左图右文**结构，条内信息是**纯垂直堆叠无分栏**。

## 布局结构

```
┌──────────────────────────────────┐
│       全屏黑色遮罩               │
│  ┌─────────────────────────┐    │
│  │ ┌──────┬──────────────┐ │    │
│  │ │      │  信息文字     │ │    │
│  │ │ 图   │  属性数据     │ │    │
│  │ │ 片   │  奖励展示     │ │    │
│  │ │      │              │ │    │
│  │ └──────┴──────────────┘ │    │
│  │   [按钮]               │    │
│  └─────────────────────────┘    │
└──────────────────────────────────┘
```

## 三步判断法（5样本+1验证，盲测1/1通过）

### 第1步：是不是横向长条面板 + 遮罩？

- 全屏黑色遮罩 + 横向长条弹窗 → 继续
- 否 → 不是展示横条系列

### 第2步：内部是不是左图右文？

- 左侧：图片/立绘/头像/物品展示位
- 右侧：文字信息/数据/奖励
- 是 → 继续

### 第3步：命名是否为 Succ/Result 结尾？

- 15个UI全以 `Succ`、`Result` 结尾（升级/结算/获得）
- 是升级/结算类 → **展示-横条-左图-右信息** ✓

---

## 常见混淆

### vs 展示-横条-条内信息

| 特征 | 左图-右信息 | 条内信息 |
|------|-----------|---------|
| 内部结构 | **左图右文**分栏 | 垂直堆叠，无分栏 |
| 典型场景 | 战斗结果/升级成功 | 奖励结算/物品获得 |
| 左区 | 必有（图片占位） | 无 |

---

## 判断口诀

> **横条遮罩结算弹，左图右文分两半。Succ Result名后缀，升级获得即此般。**
