# 木屏风+横列表

## 定义

该布局通过**木屏风+横列表**的空间组织方式，实现内容的有效展示和交互。

## 布局结构

```
┌──────────────────────────┐
│[Item] [Item] [Item] ...│ → 可横向滚动
└──────────────────────────┘
```

## 主要特征

- 清晰的空间分区方式
- 合理的内容排列逻辑
- 展示图文类型的特征表现

## 包含的 UI（共 3 个）

- [[prefab_final_analysis/UI_WishMakingExchange|UI_WishMakingExchange]]
- [[prefab_final_analysis/UI_RechargeMain|UI_RechargeMain]]
- [[prefab_final_analysis/UI_LimitExchange|UI_LimitExchange]]


## 横竖屏适配要点

- 布局在横竖屏转换时需要调整各区域的宽高比
- 关键信息应优先保证可见性
- 可根据屏幕方向调整内容的流向方式
