# 🎨 UI 布局分类工具使用指南

**文件**: `ui_layout_classifier.html`  
**功能**: 交互式 UI 界面布局分类和筛选工具  
**状态**: ✅ 完全就绪

---

## 🚀 快速开始

### 1. 打开网页
在浏览器中打开 `ui_layout_classifier.html` 文件（推荐使用 Chrome、Firefox 或 Edge）

### 2. 开始分类
- 所有 910 个 UI 界面都会显示
- 初始分成 61 个"Group"（按字母顺序）
- 每个 Group 包含 ~15 个 UI 截图

### 3. 进行筛选
- **点击 checkbox** 选中相同布局的 UI
- **鼠标悬停** 查看 UI 截图预览
- **拖拽重组** 整理相同布局的界面

### 4. 导出结果
- 点击底部的 **"查看结果"** 查看分类结果
- 点击 **"导出 JSON"** 或 **"导出 CSV"** 下载分类数据

## 🎯 功能详解

### 工具栏功能

| 功能 | 说明 |
|------|------|
| **搜索框** | 快速搜索 UI 名称 |
| **筛选器** | 按布局类型筛选（可自定义） |
| **全选** | 一键选中所有 UI |
| **取消全选** | 一键取消所有选中 |
| **重置分组** | 恢复初始分组（清除本地数据） |

### UI 卡片

每个 UI 卡片显示：
- ✅ **checkbox** - 用于选中/取消
- 📸 **截图** - UI 实际界面预览（150px 高度）
- 📝 **UI 名称** - UI 的标识符

**交互**:
- 点击 checkbox 选中
- 鼠标悬停卡片会提升动画效果
- 选中后会有蓝色边框和背景

### 分组管理

- 每个 Group 都有独立的 checkbox
- 点击 Group 的 checkbox **全选/取消该组所有 UI**
- Group 显示包含的 UI 数量

### 数据导出

#### JSON 格式
```json
{
  "相同布局1": ["UI_Bag", "UI_Beauty", ...],
  "相同布局2": ["UI_Chat", "UI_Mail", ...]
}
```

#### CSV 格式
```
Group,UI Name
"相同布局1","UI_Bag"
"相同布局1","UI_Beauty"
"相同布局2","UI_Chat"
```

## 💾 数据管理

### 本地存储
- 分类数据自动保存到浏览器的 **localStorage**
- 关闭浏览器后仍会保留
- 不同浏览器/设备的数据不会同步

### 清除数据
- 点击 **"重置分组"** 按钮
- 或在浏览器控制台执行: `localStorage.clear()`

### 导出备份
- 定期导出 JSON 文件作为备份
- 可在多台设备间转移

## 🔧 高级功能

### 自定义分组

**方式 1: 通过网页 UI**
1. 创建新 Group（编辑源代码）
2. 手动拖拽 UI 到新 Group
3. 保存到本地存储

**方式 2: 通过 localStorage API**
在浏览器控制台执行：
```javascript
// 获取当前分类
const data = JSON.parse(localStorage.getItem('uiClassifications'));

// 修改分类
data['新的布局类型'] = ['UI_Bag', 'UI_Beauty'];

// 保存
localStorage.setItem('uiClassifications', JSON.stringify(data));
```

### 导入分类数据

1. 获取之前导出的 JSON 文件
2. 在浏览器控制台执行：
```javascript
const data = { /* 粘贴 JSON 内容 */ };
localStorage.setItem('uiClassifications', JSON.stringify(data));
location.reload();
```

## ⚠️ 常见问题

### Q: 为什么看不到截图？
**A**: 
- 检查浏览器是否允许加载本地文件
- 如果从本地打开，某些浏览器可能需要服务器环境
- 解决方案：使用 Python 启动本地服务器
  ```bash
  python -m http.server 8000
  # 然后访问 http://localhost:8000/ui_layout_classifier.html
  ```

### Q: 数据会不会丢失？
**A**: 
- 数据保存在浏览器 localStorage，关闭不会丢失
- 但**清除浏览器缓存**会删除数据
- **导出 JSON** 可以备份数据

### Q: 怎样修改分组名称？
**A**: 
- 当前版本名称由初始分组决定
- 可通过**重置分组**重新开始
- 高级用户可编辑源代码中的 Group 名称

### Q: 支持拖拽吗？
**A**: 
- 当前版本不支持拖拽
- 推荐在新 Group 中重新整理
- 编辑源代码可添加拖拽功能
