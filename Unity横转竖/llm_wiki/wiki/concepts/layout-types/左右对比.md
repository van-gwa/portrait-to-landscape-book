# 左右对比

## 定义

「左右对比」是左右布局中的**对比/比较**变体——左右两列展示两个对象或同一对象的前后状态，中间常有箭头/VS标识。用于属性对比、角色互换、升级前后等场景。共8个UI。

**与「展示-横条-左右对比」的区别**：本类型是全屏/大面板对比，后者是横条弹窗对比。

## 布局结构

```
┌──────────┬──┬──────────┐
│          │  │          │
│  左对象  │VS│  右对象  │
│  (前/左) │→│ (后/右) │
│          │  │          │
└──────────┴──┴──────────┘
```

## 三步判断法（4样本+1验证，通过）

### 第1步：是不是左右并列两区？

- 左右两列，结构对称或近似对称 → 继续

### 第2步：主题是不是对比/比较/互换？

- 对比属性、升级前后、角色互换、选择对比 → 继续

### 第3步：排除横条弹窗对比

- 是横条弹窗 → 展示-横条-左右对比
- 是面板/全屏对比 → **左右对比** ✓

---

## 判断口诀

> **左右对称两区排，对比互换箭头在。非横条弹全屏比，左右对比此型来。**
