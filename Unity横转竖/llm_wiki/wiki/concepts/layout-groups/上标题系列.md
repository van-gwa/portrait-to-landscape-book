# 上标题系列

## 说明

该大类包含 **4 种布局**，共 **162 个 UI**。

## 包含的布局类型

| 布局名称 | UI 数量 |
|---------|--------|
| [上标题-中竖列表-下信息](llm_wiki/wiki/concepts/layout-types/上标题-中竖列表-下信息) | 92 |
| [上标题-中横列表-下信息](llm_wiki/wiki/concepts/layout-types/上标题-中横列表-下信息) | 57 |
| [上标题-中上标签-中竖列表-下信息](llm_wiki/wiki/concepts/layout-types/上标题-中上标签-中竖列表-下信息) | 11 |
| [上标题+下横列表](llm_wiki/wiki/concepts/layout-types/上标题+下横列表) | 2 |


**总计**：4 种布局，162 个 UI

## 大类特征

上标题系列类型的布局具有以下共同特征：
- 统一的视觉风格和交互逻辑
- 相似的空间分区方式
- 一致的信息展示策略
