# T型布局

## 说明

该大类包含 **2 种布局**，共 **54 个 UI**。

## 包含的布局类型

| 布局名称 | UI 数量 |
|---------|--------|
| [T型-上标题-左标签-右竖列表](llm_wiki/wiki/concepts/layout-types/T型-上标题-左标签-右竖列表) | 28 |
| [T型-上标题-左-右](llm_wiki/wiki/concepts/layout-types/T型-上标题-左-右) | 26 |


**总计**：2 种布局，54 个 UI

## 大类特征

T型布局类型的布局具有以下共同特征：
- 统一的视觉风格和交互逻辑
- 相似的空间分区方式
- 一致的信息展示策略
