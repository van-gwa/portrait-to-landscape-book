## 🚀 如何在 Obsidian 中查看

### 方法 1：直接打开 UI 笔记
1. 打开任何 UI 笔记（如 [[UI_Achievement]]）
2. 滚动到末尾的 **## Screenshot** 部分
3. 图片会直接显示在笔记中

### 方法 2：使用图表视图
- Ctrl+G 打开图表视图
- 搜索任意 UI 名称
- 查看其关联的资源和其他笔记

### 方法 3：使用文件搜索
- Ctrl+P 打开快速命令
- 输入 UI 名称（如 "UI_Bag"）
- 打开笔记，查看截图
