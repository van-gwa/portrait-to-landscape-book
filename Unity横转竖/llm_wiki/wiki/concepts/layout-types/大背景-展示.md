# 大背景-展示

## 定义

该布局通过**大背景-展示**的空间组织方式，实现内容的有效展示和交互。

## 布局结构

```
┌────────────────────────────┐
│                            │
│    大背景区域              │
│    ┌──────────────────┐   │
│    │   内容展示       │   │
│    │   [内容]        │   │
│    └──────────────────┘   │
│                            │
└────────────────────────────┘
```

## 主要特征

- 清晰的空间分区方式
- 合理的内容排列逻辑
- 展示图文类型的特征表现

- 布局在横竖屏转换时需要调整各区域的宽高比
- 关键信息应优先保证可见性
