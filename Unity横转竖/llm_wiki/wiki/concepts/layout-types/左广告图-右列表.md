# 左广告图-右列表

## 定义

「左广告图-右列表」是左右布局中的**大图展示**变体——左侧大幅广告图/插画/立绘展示位 + 右侧列表/详情信息。与「左-右」的核心区别：左区是**宣传性大图**而非功能性展示。共13个UI。

## 布局结构

```
┌────────────┬──────────────────┐
│            │  列表项 1        │
│  大幅广告  │  列表项 2        │
│  图/立绘   │  列表项 3        │
│  (约1/2)  │  ...             │
│            │                  │
└────────────┴──────────────────┘
```

## 三步判断法（5样本+1-2验证，通过）

### 第1步：左侧是不是大幅广告图/立绘展示位？

- 左区约占50%宽度，是大图展示/占位 → 继续
- 左区是普通图标/小图 → 考虑左-右

### 第2步：右侧是不是列表？

- 右侧为列表/记录/条目排列 → 继续

### 第3步：排除艺术设计类型

- 无彩色艺术装饰（祥云/梅花/流苏）→ **左广告图-右列表** ✓
- 有艺术装饰 → 左艺术设计+左-右

---

## vs 左-右

| 特征 | 左广告图-右列表 | 左-右 |
|------|--------------|-------|
| 左区 | **大幅广告图** | 功能性展示（缩略图/列表） |
| 左区占比 | ~50% | 不定 |

---

## 判断口诀

> **左半大图广告展，右半列表信息显。无花无云无流苏，广告右列即此篇。**
