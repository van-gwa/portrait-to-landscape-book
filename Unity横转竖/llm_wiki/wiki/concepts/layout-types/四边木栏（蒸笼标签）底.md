# 四边木栏（蒸笼标签）底

## 定义

该布局通过**四边木栏（蒸笼标签）底**的空间组织方式，实现内容的有效展示和交互。

## 布局结构

```
╔════════════════════════╗
║   ♦ 木栏框架 ♦        ║
║   ┌──────────────┐ ║
║   │   内容       │ ║
║   │   [Item]    │ ║ ↕
║   │   [Item]    │ ║
║   └──────────────┘ ║
╚════════════════════════╝
```

## 主要特征

- 清晰的空间分区方式
- 合理的内容排列逻辑
- 四边木栏系列类型的特征表现

## 横竖屏适配要点

- 布局在横竖屏转换时需要调整各区域的宽高比
- 关键信息应优先保证可见性
- 可根据屏幕方向调整内容的流向方式
