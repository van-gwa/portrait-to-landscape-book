# 左广告图-右列表

## 定义

该布局通过**左广告图-右列表**的空间组织方式，实现内容的有效展示和交互。

## 布局结构

```
┌──────────┬────────────┐
│          │            │
│   左区   │   右区     │
│          │            │
│          │            │
└──────────┴────────────┘
```

## 主要特征

- 清晰的空间分区方式
- 合理的内容排列逻辑
- 左右布局类型的特征表现
