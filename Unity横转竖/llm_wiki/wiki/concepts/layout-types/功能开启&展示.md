# 功能开启&展示

## 定义

该布局通过**功能开启&展示**的空间组织方式，实现内容的有效展示和交互。

## 布局结构

```
┌──────────────────────┐
│    布局内容区      │
│    [item 1]       │
│    [item 2]       │ ↕ 可滚动
│    [item 3]       │
└──────────────────────┘
```

## 主要特征

- 清晰的空间分区方式
- 合理的内容排列逻辑
- 弹框类类型的特征表现
