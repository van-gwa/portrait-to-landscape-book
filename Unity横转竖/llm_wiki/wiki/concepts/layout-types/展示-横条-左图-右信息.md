# 展示-横条-左图-右信息

## 定义

该布局通过**展示-横条-左图-右信息**的空间组织方式，实现内容的有效展示和交互。

## 布局结构

```
┌──────────────────────────┐
│ ┌─────┬─────────────────┐ │
│ │     │  Item 信息       │ │
│ │ 图  ├─────────────────┤ │ ↕ 可竖向滚动
│ │     │  描述内容        │ │
│ │     │                 │ │
│ └─────┴─────────────────┘ │
└──────────────────────────────┘
```

## 主要特征

- 清晰的空间分区方式
- 合理的内容排列逻辑
- 展示横条系列类型的特征表现
