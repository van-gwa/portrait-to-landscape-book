# 布局分类快速参考

> **新功能！** 962 个 UI 已按照 62 种布局类型自动分类，方便快速查找相同布局的 UI。

---

## 🚀 开始使用

### 第 1 步：打开索引
打开 [[UI_Layout_Classification_Index.md]]，你会看到所有 62 个布局类型的表格，**按 UI 数量从多到少排列**。

### 第 2 步：找到你关心的布局类型
表格按数量排序，可以快速找到：
- **最常见的布局** （如 tips - 100 UIs）
- **你要优化的特定布局** （点击即跳转）

### 第 3 步：查看该布局的所有 UI
点击任意布局类型的链接，打开该类型的笔记。笔记会显示：
- 📊 该类型包含的 UI 总数
- 📋 所有 UI 的完整列表（带 Wiki 链接）

### 第 4 步：跳转到具体 UI
点击任意 UI 链接，直接打开该 UI 的详情笔记，查看：
- 🖼️ UI 截图
- 📦 所有 Image/RawImage 资源
- 📋 完整的组件清单

## 🎯 常见使用场景

### 场景 1：我想优化 tips 类型的 UI
1. 打开 [[tips]]
2. 看到有 100 个 UI 用了这种布局
3. 逐个检查这些 UI，统一优化样式或性能
4. 批量更新相关资源

### 场景 2：我想找"确认弹框"这种布局的所有 UI
1. 打开 [[UI_Layout_Classification_Index.md]]
2. 在表格中找到"确认弹框"（87 个 UI）
3. 点击链接打开 [[确认弹框]]
4. 查看所有 87 个确认弹框 UI
5. 点击任意 UI 查看详情（截图、资源清单等）

### 场景 3：我想了解某个 UI 用的是什么布局
1. 使用 `Ctrl+P` 搜索 UI 名称（如 `UI_Bag`）
2. 打开该 UI 的笔记查看详情
3. 如果想找同类型的其他 UI，记下布局类型
4. 在 [[UI_Layout_Classification_Index.md]] 中搜索该布局类型
5. 点击链接查看同类型的所有 UI

### 场景 4：我想批量处理相同布局的 UI
1. 打开 [[UI_Layout_Classification_Index.md]]
2. 选择你要处理的布局类型（如 [[左-右]]）
3. 在该笔记中复制所有 UI 名称列表
4. 用这个列表进行脚本化处理、批量导出、样式统一等

## ⚡ 搜索技巧

在 Obsidian 中使用 `Ctrl+P` 快速搜索：

- 搜索布局类型名称（如 `tips`）→ 直接打开该布局的笔记
- 搜索 UI 名称（如 `UI_Bag`）→ 打开该 UI 的详情笔记
- 搜索关键词（如 `弹框`）→ 找到所有包含该关键词的布局
