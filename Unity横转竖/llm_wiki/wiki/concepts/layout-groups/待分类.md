# 待分类

## 说明

该大类包含 **3 种布局**，共 **11 个 UI**。

## 包含的布局类型

| 布局名称 | UI 数量 |
|---------|--------|
| [待确认](llm_wiki/wiki/concepts/layout-types/待确认) | 8 |
| [测试界面](llm_wiki/wiki/concepts/layout-types/测试界面) | 2 |
| [抽屉](llm_wiki/wiki/concepts/layout-types/抽屉) | 1 |


**总计**：3 种布局，11 个 UI

## 大类特征

待分类类型的布局具有以下共同特征：
- 统一的视觉风格和交互逻辑
- 相似的空间分区方式
- 一致的信息展示策略
