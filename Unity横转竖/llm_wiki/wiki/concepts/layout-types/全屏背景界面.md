# 全屏背景界面

## 定义

「全屏背景界面」是**全屏场景背景 + 少量UI叠加**的功能界面——无遮罩、无弹窗，以完整的场景视觉效果为底，UI元素直接浮于场景之上。属全屏类大类，共12个UI。

**核心判据**：全屏场景（非纯黑遮罩）+ 无弹窗框架 + UI浮层。

## 布局结构

```
┌──────────────────────────────┐
│                              │
│     全屏场景背景            │
│   (书桌/舞台/星空/展示台)   │
│                              │
│   [少量UI叠加：             │
│    返回/按钮/信息]          │
│                              │
└──────────────────────────────┘
    无遮罩！无弹窗框架！
```

## 三步判断法（4样本+1验证，通过）

### 第1步：是不是全屏场景？

- 背景是完整的视觉场景（非黑色遮罩）→ 继续
- 是黑色遮罩 → 不是全屏类

### 第2步：有没有遮罩或弹窗框架？

- 无遮罩、无弹窗边框 → 继续
- 有遮罩/弹窗框 → 不是本类型

### 第3步：UI是不是浮在场景上？

- UI元素（按钮/标签/信息）直接叠加在场景之上 → **全屏背景界面** ✓

---

## 判断口诀

> **全屏场景作背景，无罩无框景自明。UI浮层轻点缀，全屏背景即此型。**
