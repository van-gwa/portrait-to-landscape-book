# 大型组合

## 说明

该大类包含 **1 种布局**，共 **5 个 UI**。

## 包含的布局类型

| 布局名称 | UI 数量 |
|---------|--------|
| [左-中-右](llm_wiki/wiki/concepts/layout-types/左-中-右) | 5 |


**总计**：1 种布局，5 个 UI

## 大类特征

大型组合类型的布局具有以下共同特征：
- 统一的视觉风格和交互逻辑
- 相似的空间分区方式
- 一致的信息展示策略
